# ScriptDoc 🚀

**ScriptDoc** is a professional-grade Google Apps Script (GAS) add-on designed to automatically generate comprehensive, structured, and AI-enhanced documentation for your script projects directly within Google Docs.

**ScriptDoc** est un module complémentaire professionnel pour Google Apps Script (GAS) conçu pour générer automatiquement une documentation complète, structurée et améliorée par l'IA pour vos projets de scripts, directement dans Google Docs.

## ✨ Key Features | Fonctionnalités Clés

### 🇬🇧 English
- **Automated Analysis**: Scans your entire GAS project including manifest, files, and JSDoc.
- **AI Retro-Documentation**: Integrates with **Gemini 3** to automatically explain what each function does, even if it lacks comments.
- **Visual Structure**: Generates a clear, tree-like diagram of your project architecture.
- **Bilingual Interface**: Automatically adapts to your Google account language (French or English).
- **Premium UI**: Sleek, modern sidebar built with Material Design 3 principles.
- **Template Selection**: Choose between *Standard*, *Technical Deep-dive*, or *API Reference* formats.

### 🇫🇷 Français
- **Analyse Automatisée** : Scanne l'intégralité de votre projet GAS, y compris le manifeste, les fichiers et le JSDoc.
- **Rétro-documentation par IA** : Intègre **Gemini 3** pour expliquer automatiquement le rôle de chaque fonction, même sans commentaires.
- **Structure Visuelle** : Génère un diagramme en arborescence clair de l'architecture de votre projet.
- **Interface Bilingue** : S'adapte automatiquement à la langue de votre compte Google (Français ou Anglais).
- **UI Premium** : Barre latérale moderne et élégante conçue selon les principes de Material Design 3.
- **Choix de Modèles** : Choisissez entre les formats *Standard*, *Analyse Technique* ou *Référence API*.

## 🛠️ Tech Stack | Technologies

- **Backend**: Google Apps Script (V8 Runtime).
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6).
- **APIs**: 
  - Google Apps Script API (Content retrieval).
  - Google Drive API (Metadata).
  - Google Picker API (Project selection).
  - Gemini AI v1beta API (Retro-documentation).

## 🚀 Installation & Setup | Installation & Configuration

### 🇬🇧 English
1. **Deploy**: Create a new Google Doc and open **Extensions > Apps Script**. Copy the files from this repository.
2. **Enable APIs**: Turn **ON** the Google Apps Script API in your [Apps Script Settings](https://script.google.com/home/usersettings).
3. **Standard Project**: Link your script to a **Standard Google Cloud Project** in the Project Settings.
4. **GCP Console**: Enable the **Google Apps Script API** and **Google Picker API** in your Cloud Console.
5. **Usage**: Refresh your Doc, open the **ScriptDoc** menu, and select your project.

### 🇫🇷 Français
1. **Déploiement** : Créez un Google Doc, allez dans **Extensions > Apps Script** et copiez les fichiers de ce dépôt.
2. **Activer les APIs** : Activez l'API Google Apps Script dans vos [Paramètres Apps Script](https://script.google.com/home/usersettings).
3. **Projet Standard** : Liez votre script à un **Projet Google Cloud Standard** dans les paramètres du projet.
4. **Console GCP** : Activez l'**API Google Apps Script** et l'**API Google Picker** dans votre console Cloud.
5. **Utilisation** : Actualisez votre Doc, ouvrez le menu **ScriptDoc** et sélectionnez votre projet.
